    Project: Make a website

      Create a one-page website on a topic of your choice. Your website should have the following features:
      1. All HTML structural markup is correct. Be sure tags are all opened, closed, and nested properly.
      2. Content should be well thought out, organized, and all related to some overall topic.
      3. Write 3-4 paragraphs.
      4. Include 2-3 images, each scaled appropriately.
      5. Include 2-3 hyperlinks.
      6. Include some other HTML data structure, perhaps a list or table.
      7. Style the page nicely with colors and backgrounds.
      8. Use Google fonts.
      9. Take steps to make sure content is easy to read on large screens.
      10. Make sure the site looks nice on mobile.
      To turn in, attach a link to the GitHub Pages URL for your site.
    To turn in, attach a link to the GitHub Pages URL for your site. If you already had web design, your site should include 7-8 of the above items and also demonstrate some new learning about HTML and CSS or some other web coding technology. In addition to a GitHub Pages URL, attach a Google Doc with a summary of the new content you learned and how it was applied to your website.
